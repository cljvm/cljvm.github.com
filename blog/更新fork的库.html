更新fork的库

1、把需要更新的fork库克隆到本地 git clone --depth 1 https://github.com/cljvm/shadowsocks-rust.git cd shadowsocks-rust

2、使用 git remote -v 查询远程仓库 git remote -v origin https://github.com/cljvm/shadowsocks-rust.git (fetch) origin https://github.com/cljvm/shadowsocks-rust.git (push)

3、配置一个上游仓库 git remote add upstream https://github.com/shadowsocks/shadowsocks-rust.git

4、确认刚才添加的远程仓库 git remote -v origin https://github.com/cljvm/shadowsocks-rust.git (fetch) origin https://github.com/cljvm/shadowsocks-rust.git (push) upstream https://github.com/shadowsocks/shadowsocks-rust.git (fetch) upstream https://github.com/shadowsocks/shadowsocks-rust.git (push)

8、更新到 GitHub 的 fork 上 git push origin master
